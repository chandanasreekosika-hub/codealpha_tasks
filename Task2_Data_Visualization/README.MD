# Task 2 - Superstore Data Visualization
This project analyzes Superstore sales data and creates visualizations using Python.
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
## Project Files
- data_visualization.ipynb
- data_visualization.html
- Sample - Superstore.csv
- category_sales.png
- region_sales_pie.png
- profit_by_category.png
- sales_vs_profit.png
- top10_subcategories.png
- correlation_heatmap.png
## Visualizations
1. Total Sales by Category
2. Sales Distribution by Region
3. Profit by Category
4. Sales vs Profit Scatter Plot
5. Top 10 Subcategories by Sales
6. Correlation Heatmap
## Key Insights
- Technology generated the highest sales.
- The West region contributed the highest sales.
- Some high-sales products generated losses.
- A strong relationship exists between sales and profit.

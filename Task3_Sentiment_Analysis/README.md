# Task 3 - Amazon Reviews Sentiment Analysis
This project performs sentiment analysis on Amazon product reviews using Python.
## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- WordCloud
## Project Files
- sentiment_analysis.ipynb
- sentiment_analysis.html
- Amazon_Reviews.csv
- sentiment_distribution.png
- rating_distribution.png
- wordcloud.png
## Conclusion
Most reviews were positive, indicating high customer satisfaction. Positive words such as "Great", "Good", "Best", and "Love" appeared frequently in customer reviews.
